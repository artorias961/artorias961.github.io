Introduction

Computation or class EE 5450 started with the introduction to Python. Where we will learn the syntax and module (library) loading system. In doing so, we will learn how to install the basic programs that will be used and understand the basics of python before focusing what is Computation. First, we are going to start of by installing Miniforge. Where Miniforge is a conda-based Python distribution that includes a minimal set of packages necessary to run Python. It is intended to be a lightweight alternative to the full Anaconda distribution, and is a standaline distribution.
Once you have installed Miniforge completely, open Miniforge prompt input the following to create a environment: 

                    
                    
                        
                         mamba create -n your_environment_name

Once we created the environment, we are going to open the environment by applying the following commands: 

                    
                    
                        
                         mamba activate your_environment_name

Now we are inside the environment, we are going to install libraries by doing the following command: 

                    
                    
                        
                         mamba install pandas matplotlib sklearn

Once you have installed all the required libraries, then you will need to go to the same environment where we need to open Jupyter Lab. We do this by applying the following code: 

                    
                    
                        
                         jupyter lab

Now you should have everything you need for the Computation. It should be mentioned that these will be my notes for future references and assuming you know Python already.
